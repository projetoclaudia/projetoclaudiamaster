# C.L.A.U.D.I.A
## Controle Lógico de Acessos Unificado Dinâmico com Inteligência Artificial

## Visão geral
Plataforma de gestão de identidades e acessos (IAM) que combina:
- Controle de usuários, papéis e permissões (RBAC)
- Auditoria completa de eventos
- Camada de IA para recomendações de acesso e detecção de anomalias
- Integração com os principais provedores de segurança e identidade do mercado

## Stack tecnológica (frontend)
- React 18 + TypeScript 5
- Vite – build ultrarrápido
- Tailwind CSS + shadcn/ui – UI moderna e acessível
- Zustand – gerenciamento de estado leve
- React Hook Form + Zod – validação tipada
- Axios – cliente HTTP com interceptors
- React Router v6 – roteamento declarativo
- pnpm – gerenciador de pacotes recomendado

## Como rodar o projeto (Ubuntu 24.04 LTS)

  pnpm install
  pnpm dev
